# qa-engineering-lab

![Playwright Tests](https://github.com/abdulosmande/qa-engineering-lab/actions/workflows/playwright.yml/badge.svg)
![API Tests](https://github.com/abdulosmande/qa-engineering-lab/actions/workflows/api-tests.yml/badge.svg)
![Accessibility](https://github.com/abdulosmande/qa-engineering-lab/actions/workflows/accessibility.yml/badge.svg)

`qa-engineering-lab` is an actively maintained portfolio project demonstrating modern QA engineering practices — ranging from end-to-end testing and API automation to accessibility and contract testing. It is being developed as part of a structured certification and hands-on learning roadmap (ISTQB, IREB, CompTIA) and serves as evidence of practical implementation capabilities in software testing, test management, and test automation projects.

## Tech Stack

| Tool                    | Domain                       | Status  |
| ----------------------- | ---------------------------- | ------- |
| Playwright + TypeScript | E2E & API Testing            | Active  |
| Postman / Newman        | API Testing                  | Planned |
| Karate                  | BDD API Testing              | Planned |
| Pact                    | Contract Testing             | Planned |
| axe-core / Lighthouse   | Accessibility Testing        | Planned |
| Selenium 4 + Java       | UI Automation (Legacy Skill) | Planned |
| GitHub Actions          | CI/CD                        | Active  |

## Background

This repository accompanies a multi-month learning and certification roadmap in software testing and test management (ISTQB CTAL series, ISTQB Specialist certifications, IREB CPRE, CompTIA A+). Each module is developed alongside the corresponding practical learning phases and continuously expanded over time.
